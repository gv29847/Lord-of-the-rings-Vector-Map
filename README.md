# Lord-of-the-rings-Vector-Map
Lord of the rings Vector Map in Corel Draw. Contoured, indexed, multi layer allows for wide range of sizes.
Font Whiffy is provided and must be installed into Windows. Download the whiffy.ttf file, double click and click Install when the popoup popsup. Do this before Running Corel Draw.
Created and opens in Draw 8 - probably on Windows 7, and in Draw 21 on Windows 11.
